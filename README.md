# Akan Name Generator

## Description
This is a web application that takes a user's birthday and gender, calculates the day of the week they were born, and then generates their traditional **Akan name**. 

Akan names are derived from the culture of the Akan people in Ghana. People are often named after the day of the week they were born.

## Features
* Calculates the exact day of the week from any date of birth.
* Provides gender-specific Akan names for both Male and Female.
* Clean, responsive user interface.
* Instant results display.

## Technologies Used
* **HTML5** - For the structure of the application.
* **CSS3** - For styling and layout (Flexbox).
* **JavaScript** - For the date calculation logic and DOM manipulation.
